# Utils

Si estoy en la rama 'MCV' y quiero hacer enviarla a main, entonces:
``` bash 
git add .
git commit -m "AÑAÑIN"
# Enviar los cambios al repositorio pero no a la rama main
git push origin MVC
# Cambiar y actualizar la rama main
git checkout main
git pull origin main
# Unir las ramas
git merge MVC
```
